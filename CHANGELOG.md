# Changelog

## v0.2 — 2026-06-15

### Нови функции
- Полето "Единица мярка" в продуктовата форма е заменено с падащо меню (бр., кг., г., л., мл., м., см., кутия, кашон, пакет, бутилка)
- В полето "Количество" при движение се показва единицата на избрания продукт автоматично
- BG/EN езиков превключвател в страничното меню — пълен превод на целия интерфейс

### Поправки
- OUT движение не позволява количество над наличното — показва се ясно потребителско съобщение
- TRANSFER движение не позволява количество над наличното в изходната локация
- Количеството не може да бъде 0 или отрицателно (валидация на клиент и сървър)
- Server Components crash при невалидно движение е премахнат — вместо crash се показва inline error съобщение
- Server action `submitMovement` връща структуриран резултат `{ success, error }` вместо да хвърля exception
- Добавена защита срещу отрицателна наличност на сървърно ниво (проверка преди RPC)
- Колоната "Забележка" в историята на движенията вече показва пълния текст (без отрязване)
